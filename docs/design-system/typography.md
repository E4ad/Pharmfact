# Typographie

Pharmfact utilise Inter et une échelle modulaire basée sur un ratio 1.25.

| Token | Taille | Usage |
| --- | --- | --- |
| `xs` | 12px | Captions et aides secondaires |
| `sm` | 14px | Métadonnées, labels, texte secondaire |
| `base` | 16px | Texte principal |
| `lg` | 20px | Sous-titres courts |
| `xl` | 25px | Titres de cartes |
| `2xl` | 31px | Sections |
| `3xl` | 39px | Titres de pages compacts |
| `4xl` | 49px | Titres principaux |

## Règles

- Garder le texte principal à 16px minimum.
- Limiter chaque vue à 3 ou 4 tailles visibles.
- Utiliser `fontWeights.semibold` pour les labels et titres de cartes.
- Réserver les grands titres aux pages, pas aux composants internes.
