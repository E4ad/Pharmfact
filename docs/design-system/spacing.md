# Espacements

L’échelle est basée sur une unité de 4px.

| Token | Valeur | Usage typique |
| --- | --- | --- |
| `2xs` | 4px | Micro-ajustements |
| `xs` | 8px | Petits gaps |
| `sm` | 12px | Cellules et densité compacte |
| `md` | 16px | Espacement standard |
| `lg` | 24px | Padding de carte |
| `xl` | 32px | Sections internes |
| `3xl` | 48px | Séparation de sections |

## Règles

- Utiliser `contextSpacing` pour cartes, formulaires, tables et modales.
- Éviter les valeurs ad hoc comme `17px` ou `22px`.
- Préférer une densité plus compacte dans les tables et plus aérée dans les formulaires.
