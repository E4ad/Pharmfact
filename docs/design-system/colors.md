# Couleurs

## Marque

Le bleu principal Pharmfact reste `#2563eb`, exposé via `brandColors.primary[600]`.

| Token | Valeur | Usage |
| --- | --- | --- |
| `brandColors.primary[600]` | `#2563eb` | Boutons et actions primaires |
| `brandColors.primary[700]` | `#1d4ed8` | Hover/active light mode |
| `brandColors.primary[400]` | `#60a5fa` | Surfaces sombres ou accents doux |

## Sémantique

| Famille | Usage |
| --- | --- |
| `success` | Succès et états complétés |
| `warning` | Avertissements récupérables |
| `error` | Erreurs et actions destructives |
| `info` | Information neutre |

## Neutres

L’échelle `neutralColors.slate` sert aux textes, bordures et surfaces. Elle donne un gris légèrement bleuté, plus adapté à une application médicale qu’un gris strictement neutre.

## Règles

- Ne pas ajouter de couleurs hexadécimales dans un composant sans créer ou réutiliser un token.
- Vérifier le contraste dès qu’un token est utilisé comme texte.
- Ne jamais communiquer un état uniquement par la couleur.
